# OGE Informatics Cheat Sheet

Статический сайт-шпаргалка по ОГЭ по информатике.

## Рекомендованный репозиторий

Создай отдельный репозиторий с именем `oge-informatika`.

Тогда сайт будет доступен по адресу:

`https://violenceeke.github.io/oge-informatika/`

Корневой сайт `https://violenceeke.github.io/` останется твоим портфолио.

## Что уже готово

- основной файл сайта: `index.html`
- адаптация под GitHub Pages
- workflow для автодеплоя через GitHub Actions
- `.nojekyll` для безопасной публикации статического сайта

## Как опубликовать

1. Создай на GitHub новый репозиторий `oge-informatika`.
2. Привяжи его как `origin`.
3. Запушь ветку `main`.
4. В настройках репозитория открой `Settings -> Pages`.
5. Убедись, что источник публикации настроен на `GitHub Actions`.

После первого пуша workflow сам выложит сайт на GitHub Pages.
